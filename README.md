# 💰 Expense Tracker Using Python

## Overview
Expense Tracker is a Python-based console application that helps users record and manage daily expenses. Users can add expenses, view records, calculate total spending, and search expenses by category. This project is ideal for beginners to learn Python fundamentals such as functions, lists, dictionaries, loops, conditional statements, and user input handling.

##  Features
✔ Add Expense  
✔ View All Expenses  
✔ Calculate Total Expenses  
✔ Search Expenses by Category  
✔ Menu-Driven Interface  
✔ Beginner-Friendly Project  
✔ Easy to Upgrade with File Handling, Database, and GUI

##  Technologies Used
- Python 3
- Lists
- Dictionaries
- Functions
- Loops & Conditional Statements

##  Project Workflow
1. User selects an option from the menu.
2. Expense details (Date, Category, Amount, Description) are entered.
3. Expense is stored as a dictionary inside a list.
4. Users can view, search, and calculate expenses.
5. Program runs until the user exits.

##  How to Run
```bash
python expense_tracker.py
```
